# 🚀 Advanced 3D Animated Portfolio
 
A premium, production-level 3D animated portfolio website built with React, Three.js, GSAP, and Framer Motion.
 
## ✨ Features
 
- **3D Graphics**: Interactive 3D elements with Three.js
- **Smooth Animations**: GSAP timeline-based animations & ScrollTrigger
- **Page Transitions**: Framer Motion for elegant page transitions
- **Responsive Design**: Mobile-first, works on all devices
- **Performance Optimized**: Code splitting, lazy loading, GPU acceleration
- **Smooth Scrolling**: Lenis for butter-smooth scroll experience
- **State Management**: Redux Toolkit for global state
- **Custom Cursor**: Interactive cursor following mouse
- **Accessibility**: WCAG compliant with reduced motion support
 
## 🛠️ Tech Stack
 
- **Frontend**: React 18 + React Router
- **Styling**: Tailwind CSS + CSS Modules
- **Animation**: GSAP + Framer Motion
- **3D Graphics**: Three.js + React Three Fiber
- **State**: Redux Toolkit
- **Build**: Vite
- **Scroll**: Lenis
